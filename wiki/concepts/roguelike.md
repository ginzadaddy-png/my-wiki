매 플레이("런")마다 레벨·아이템·적 배치가 절차적으로 재구성되고, 죽으면 런을 처음부터 다시 시작하는 장르. 원조 로그(Rogue, 1980) 이후 정립됐다. [[soulslike|소울라이크]]가 *수공예 공간 + 공정한 난이도*라면, 로그라이크는 *절차적 생성 + 빌드 다양성*으로 반복 플레이성을 만든다.

## 장르 정의 요소

| 요소 | 내용 |
|------|------|
| 절차적 생성 | 런마다 맵·드롭·적 배치가 달라져 매번 새로운 판 |
| 런 기반 구조 | 1회차가 곧 단위 — 죽으면 그 런은 소멸 |
| 퍼머데스 | 사망 시 진행 리셋 (해당 런 한정) |
| 빌드·시너지 | 런 도중 획득 조합으로 매번 다른 전략 |
| 메타 진행성 | 런 사이에 영구 강화 누적 (= "로그라이트") |

## 로그라이크 vs 로그라이트

순수 로그라이크는 런 간 누적이 거의 없는 반면, 현대 상업작 대부분은 **메타 진행성**을 둔 **로그라이트**다. 영구 강화·해금이 죽음의 좌절을 진척감으로 전환해 진입장벽을 낮춘다. [[kasavin-hades-ea|하데스(슈퍼자이언트)]]는 영구 진행성 + 내러티브 동기로 *"신이 아니어도 즐길 수 있게"* 난이도 곡선을 완화했다.

## 핵심 설계 철학

### "한 판 더" 루프
절차적 생성 + 빌드 다양성이 결합하면 매 런이 달라져 반복이 지루하지 않다. [[balatro|발라트로]]는 조커 시너지 빌드만으로 *"한 번 잡으면 멈추지 못하는 루프"*를 만들어, 미려하지 않은 비주얼에도 게임성 자체가 셀링 포인트가 됐다.

### 죽음 = 리셋이지만 진척감 보존
소울라이크의 죽음이 *학습 피드백*이라면, 로그라이크의 죽음은 *판의 리셋*이다. 메타 진행성·내러티브가 이 리셋을 좌절이 아닌 누적으로 바꾼다.

### 내러티브와의 결합 (하데스 모델)
[[kasavin-hades-ea|하데스]]는 런 반복을 *서사 장치*로 역이용했다 — 죽고 돌아올 때마다 캐릭터 관계·대사가 갱신돼, 반복이 곧 스토리 진행이 된다. [[early-access-strategy|얼리 액세스]]를 "파일럿 에피소드"처럼 운영해 캐릭터 단위로 톤을 검증한 사례.

## 절차적 생성 — 완전 무작위 vs 하이브리드

완전 무작위 생성은 가능성 공간 전체(P=T^B, T=타일 유형·B=그리드 크기)를 열지만 결과물 대부분이 통과 불가·가독성 결여라, 현대 로그라이크는 *핸드메이드 + 알고리즘 제약*의 **하이브리드**를 쓴다.

- **Spelunky**: 4×4 그리드에 끊기지 않는 '해결 경로(Solution Path)'를 먼저 보장한 뒤 외곽을 절차적으로 채움
- **Dead Cells**: 개념 그래프(Concept Graph)로 레이아웃을 선제 통제하고, 타일당 *스폰 예산*으로 몬스터 공정성 확보
- **Hades**: 방 지형은 100% 수공예로 고정하고 알고리즘은 *방 선택 + 적·보상 결합*만 담당 → AAA급 연출·조작감 보전

상세 대비는 [[pcg-pure-vs-hybrid|완전 절차생성 vs 하이브리드]] 비교 페이지 참조. AAA 3D에서는 격자 그리드 자체가 깨져서, [[gdc22-returnal-procedural-world|Returnal]]은 그리드를 포기하고 자유형 레이아웃 + 수작업 공간 제약으로 갔고 런 길이·세이브 같은 장르 규칙까지 수정해야 했다.

## 난이도 사다리와 의도적 불균형

메타 진행성은 진입장벽을 낮추지만 마스터리 가치를 훼손할 위험이 있어, 명작은 **난이도 사다리**로 완충한다. [[gdc19-slay-the-spire-metrics|Slay the Spire]] 승천(Ascension)은 엘리트 빈도↑ → 저주 카드 강제(A10) → 상점 물가↑(A15) → 이중 보스 검증(A20)으로 *런 이코노미 규칙*을 건드린다. 하데스 형벌 규약은 플레이어가 페널티를 직접 조합하는데, *개발사 공식 테스트(Heat 40·보상 종결 32)와 커뮤니티 역산 한계(Heat 64)를 구분*하는 게 중요하다.

빌드 설계의 핵심은 **평균적 밸런싱 거부**다. 강력한 시너지(StS Corruption 무한 엔진 등)를 *허용하되 동시 출현 확률을 희소화*해 매 판 재현을 막고, [[data-driven-development|메트릭]](픽률·승률)으로 아웃라이어를 메커니즘 삭제가 아닌 *상황적 페널티*로 유기적으로 다듬는다.

## 시장 분포 — 신규 진입 친화 장르

[[carless-genres-ruled-steam-2025-06|Carless 장르 taxonomy]] 기준, **Action Roguelike는 $1M+ 달성작 104개**이며 그중 *70개 이상이 2020년 이후 출시*다. 평균 매출은 낮지만 분포가 **healthy**(메가히트 독점 없음)해, Arena Shooter·Battle Royale 같은 winner-take-all 장르와 달리 *인디·중소의 신규 진입 가능성이 큰* 장르로 분류된다.

> 💡 **핵심 인사이트:** 로그라이크의 본질은 난이도가 아니라 **재생성(re-generation)**. 절차적 생성·빌드 시너지·메타 진행성이 결합해 "매번 다른 한 판"을 값싸게 양산하고, 이것이 인디가 적은 예산으로 긴 플레이타임과 healthy한 시장 분포를 동시에 얻는 이유다.
