게임의 출시와 마케팅을 성공시키기 위한 전략. 스팀 플랫폼 전략부터 통합 마케팅 채널 운영, 트레일러, 크리에이터, PR까지 포괄한다. 데이브 더 다이버(민트로켓), SGC 패널, Dotemu, ICO Partners 등 실전 사례 기반.

**스토어 페이지**
- 개발과 동시에 설계하는 첫 마케팅 무대 (출시 후 꾸미는 전시 공간이 아님)
- 스크린샷 한 장으로 게임을 알 수 있어야 함. "즉각 이해되는 메시지"
- 첫 트레일러: 인트로 짧게, 플레이 장면 빠르게

**태그 전략**
- 유저 언어 사용, 성공작 태그 벤치마킹
- 너무 광범위 → 거대 장르에 묻힘 / 너무 희귀 → 유입 없음
- 개발 규모에 맞는 경쟁 강도 선택 (인기 태그 = 인원·예산 부담)

**위시리스트와 팔로워**
- 위시리스트 = "밸브 달러": 교섭력, 추천 알고리즘 트리거, 이벤트 노출 우위
- 팔로워 + 뉴스 포스트 결합 시 효과 극대화
- 위시리스트 수 ≠ 판매량, 오래된 위시리스트는 효력 감소 → 주기적 관심 유도 필요

**데모와 페스티벌**
- 넥스트 페스트 전 소규모 선공개 → 피드백 반영 → 완성도 높은 버전으로 본 행사
- 대형 세일보다 장르별 소규모 이벤트에서 집중 효과가 높을 수 있음
- 출시 시기: 넥스트 페스트·장르 이벤트 직후가 이상적

**커뮤니티 운영 (데이브 더 다이버 사례)**
- Discord: 개발자 직접 답변 원칙 (못 하는 이유·일정·우선순위 투명하게)
- Reddit: 비개입 원칙, 감동적 사례에만 제한 등장
- Steam 커뮤니티: 핵심 이슈에만 개입
- 대표 명의 직접 소통이 대행사 멘트보다 신뢰를 만든다
- 밸브로부터 "커뮤니티 운영이 인상적" 피드백 획득

**플랫폼 관계**
- 쇼케이스(State of Play, Xbox Showcase 등) 슬롯 = 배너 광고보다 효율 압도
- 최소 1~2년 전부터 빌드·아트·진척 공유. 초청 시 즉시 달려가 네트워킹
- 파트너 커넥트 등 비공개 행사 최우선 참석

**가격 정책**
- 장르 평균 이하는 품질 의심 초래 역효과
- 가격 = 기대치와 이미지의 기준

> **핵심 인사이트:** 스팀 성공의 공통점은 출시일이 아닌 "출시 전부터 오랜 관계를 쌓아온 것"이다. 플랫폼, 커뮤니티, 크리에이터 모두 마찬가지.

## 데모 우선 전략 (GDC25 크리스 주코프스키)

- "모든 마케팅 활동은 데모에 비하면 미미하다"
- 단계: 데모 → 플레이테스트 → 넥스트 페스트(그랜드 피날레) → 출시
- **Real Steam(비공식)**: 출시 후 6~9개월 내 약 $15만 달러 달성 시 데일리 딜·세일 피처링 혜택
- 디스커버리 큐가 전체 트래픽 45%. 출시 첫 2일에 25,000회 노출 → 3일 째 성과가 이후 결정

## 주목의 수레바퀴 (발라트로 사례)

미디어 + 인플루언서 + 커뮤니티 세 기둥이 맞물려 돌아가는 구조.  
비주얼이 약한 게임은 데모로 경험 자체를 파는 전략이 유효.  
인플루언서 선정 기준: 구독자 수보다 **게임과의 접점** (유사 장르 플레이 이력).

## 스팀 3단계 구조 (GDC24 크리스 주코프스키)

스팀 상점은 사실상 2개 — **초기 인디** vs **레알 스팀**(상위 6%, 26만 달러 이상)

**초기 인디 핵심 팁:**
- 번역 우선순위: 중국어 → 일본어·한국어 → FIGS
- 월요일 출시 강추 (주말 3일 간 노출 유지), 화요일 금지
- AAA와 같은 날 출시 → 경쟁 피한 게임들이 빠져 슬롯 확보 가능
- 퍼블리시 버튼 꼭 누르기 (미클릭 시 8,000달러/일 기준 재산정)

**레알 스팀 진입 후:**
- 밸브 프로모션 제안 오면 **무조건 YES** (데일리 딜 25만달러, 위크엔드 딜 100만달러 기대)
- 출시 할인을 시즌 세일과 겹치게 설계 → 30일 제한 우회
- 요청해야만 해준다 — 스팀은 절대 먼저 알려주지 않는다

## 중견 스튜디오 유료 마케팅 전략 (Airbridge × Liquid Advertising)

**PC/콘솔 유저 구매 행동**
- 모바일과 달리 충동 설치 없음 — 리뷰·커뮤니티·트레일러·비교를 거쳐 구매
- "단순 노출보다 **발견의 순간** 설계가 중요"
- **문맥(Context)이 클릭보다 중요** — 잘못된 채널 노출은 브랜드 이미지 훼손

**위시리스트 질 기준**
- 최소 **70% 이상은 자발적** 위시리스트여야 함
- 이벤트 보상으로 모은 위시리스트 → 출시 전환율 실망
- 스팀은 단순 수치보다 **모멘텀**을 중시

**브랜딩 + 퍼포먼스 마케팅 통합**
- 기존 인식: PC/콘솔 = 브랜딩(트레일러·PR), 퍼포먼스 = 모바일 전용
- 라이브 서비스·얼리 액세스 확산 → PC 마케팅에도 데이터 기반 전략 필수
- 도달률, 프리퀀시, SOV(Share of Voice)를 다양한 미디어에 균형 적용

**아시아 스튜디오가 놓치는 채널 3가지**
- **커뮤니티 (Discord, Reddit)**: '안다'와 '의미 있게 존재감을 드러내는 것'은 다름
- **하이터치 프로그래매틱**: 클릭(상위 퍼널)만이 아닌 전체 퍼널 연결. 노출 순서·빈도·오디언스 타겟팅
- **OTT/CTV**: 미국·유럽에서는 틈새 아님. 삼성·LG·Vizio 시청 데이터로 광고 시청→실제 플레이 추적 가능

**크로스 플랫폼 함정**
- 콘솔 코어 유저에게 App Store 출시 광고 → "모바일 게임이구나" 인식 → 구매 의향 하락
- 해결: 오디언스 세그먼트 분리 / 플랫폼별 메시지 조정 / 메시지 분산 시 중립 톤 유지

**크리에이티브 원칙**
- 장르·USP 모호하지 않게: "슈터 게임"이 아닌 "80년대 공포 테마의 로그라이크 협동 FPS"
- AA→AAA 인식 점프는 **비주얼 '분위기'**로 전달

## 스팀 넥스트 페스트 정량 데이터 (2025–2026)

([[steam-next-fest-2026-analysis]], [[steam-next-fest-wishlist-benchmarks-2025]])

**성과 티어 (2026년 2월 기준)**
- 최하위: 1,000개 이하 / 중간: 2,000–3,000개 / 상위: ~15,000개 / 초대박: 30,000–45,000개 (행사당 3–5개)

**사전 위시리스트 vs 행사 성과** (r=0.825, 가장 강력한 예측변수)

| 사전 위시리스트 | 행사 중 중간값 |
|---------------|-------------|
| 0–999 | 322 |
| 1,000–9,999 | 1,006 |
| 10,000–99,999 | 5,215 |
| 100,000+ | 12,882 |

- **최소 2,000개** 사전 위시리스트 확보 후 입장해야 유의미한 경쟁 가능
- 데모 shadow-drop 비추천 — 행사 1개월 이상 전 미리 출시한 게임이 일관되게 우수
- 3일 차부터 노출이 상위 게임에 집중 → 초반 모멘텀이 전부

**알고리즘 변화 (2024년 10월~)**
- Valve ML 기반 가시성 알고리즘 도입 후 재보정 진행 중
- 노출 효율 대폭 개선: Crashlands 2 (620만 노출→15,987 위시리스트) → How Many Dudes? (64.5만 노출→14,740 위시리스트)
- 타겟이 명확한 게임일수록 적은 노출로 높은 위시리스트 전환 가능

## 스팀 스토어 페이지 최적화 핵심 ([[steam-page-optimization-guide-2026]])

- 트레일러 첫 5초 이내 게임플레이 시작 → **전환율 +31%** (게임플레이를 18초→1초로 이동한 사례)
- 스크린샷 8–12장 권장, 첫 번째가 검색 결과에 노출됨
- 태그 20개 전부 사용, 상위 5개가 알고리즘 가중치 최대, 범용 태그("Indie") 상위 금지
- 캡슐 아트: 최소 크기(120×45px)에서 로고 가독성 필수, 아트워크+제목만 허용
- 20% 이상 할인 시에만 위시리스트 알림 이메일 발송

## 2026년 마케팅 캘린더 ([[game-marketing-strategy-2026]])

- GTA 6 (2026년 11월 19일) 이 연도를 양분 → **Q2(4–6월)이 황금 런치 윈도우**
- SNF는 출시 전 1회만 사용 권장
- 크리에이터 중심 전략 필수: 전통 게임 미디어 인력 축소, TikTok·YouTube가 1차 채널
- 큐레이션 쇼케이스(SGF, Triple-I 등) 슬롯 = 돈으로 살 수 없는 신뢰도
