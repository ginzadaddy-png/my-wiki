게임 스튜디오가 공유하는 가치관·행동 원칙·의사결정 방식의 총체. 리더십 스타일, 원칙 문서, 팀 커뮤니케이션 방식 등을 포함한다.

## 원칙의 역할 (셸 게임즈 사례)

잘 정의된 원칙은:
- 어려운 결정의 기준점 ("원칙 X에 따르면 이게 맞다")
- 갈등의 객관적 해결 수단
- 채용 시 문화 적합성 필터
- 리더 교체 시에도 일관성 유지

## 원칙 부재의 원인 — 6가지 악마

게으름 / 자만 / 오만 / 겁쟁이 / 허영 / **경멸** (가장 위험 — "이런 거 다 허울이야")

## 원칙을 살리는 3가지 천사

- **명확성**: 팀이 실제로 이해하는지 확인
- **용기**: 불편한 대화도 감수
- **진정성**: 뼛속에서 우러나온 믿음만이 살아있는 원칙이 됨

## 5가지 질문으로 원칙 만들기

1. 왜 이 스튜디오가 존재하나? → 미션
2. 가장 중요한 건? → 핵심 가치
3. 어떻게 행동으로 옮기나? → 운영 원칙
4. 지금 집중할 것은? → 최우선 과제
5. 궁극적으로 무엇을 꿈꾸나? → 비전

→ 상세: [[gdc25-schell-studio-principles]], [[vision-statement|비전 선언문]]

## 문화를 의도적으로 설계하기 (Fisher, 2026)

[[gdc26-three-mindset-shifts|Three Mindset Shifts]]는 "훌륭한 사람들이 모이면 문화가 자연스럽게 형성된다"는 믿음을 반박한다.

**의도적 문화 설계의 증거**: 잘 작동하는 문화에서는 팀원 누구나 답할 수 있다:
- 우리의 가치관은? / 내가 내야 할 결과물은? / 어떻게 소통해야 하는가? / 의사결정은 어떻게 이루어지는가?

**팀 핸드북**: 미션·가치관, 팀 목표·제약, 커리어 성장 기대치, 협업 규칙을 문서화. 단일 오너가 최신 상태 유지 책임.

## 심리적 안전감 — 고성과 팀의 실제 비결

구글의 '아리스토텔레스 프로젝트'(2012~2015, 180여 팀 조사)는 최고의 팀을 결정하는 요인이 개인 역량이 아니라 **심리적 안전감**임을 밝혔다. ([[gamedev-leadership-dynamics|리더십 역학 보고서]], [[kumhotire-psychological-safety|심리적 안전감 뉴스레터]])

고성과 팀의 5가지 요소 (중요도 순):
1. **심리적 안전감** — 처벌·비난 걱정 없이 의견·아이디어를 솔직하게 표현할 수 있는 환경
2. 신뢰성 / 3. 구조와 명확성 / 4. 일의 의미 / 5. 일의 영향력

**심리적 안전감 작동 vs 부재의 극단적 대비:**
- **테네리페 공항 참사(1977)**: 기장의 권위 → 부기장이 문제 지적 못 함 → 583명 사망. 이후 CRM 프로그램으로 항공 업계 심리적 안전감 제도화
- **허드슨강의 기적(2009)**: 기장-부기장 신뢰 기반 신속 소통 → 155명 전원 생존

게임 스튜디오에서의 적용:
- 주니어 개발자가 낙인 두려움 없이 "모르겠다"고 말할 수 있는 환경
- 실수를 비난이 아닌 학습 기회로: Etsy 'Blameless Culture', 덴디 피플 'Biggest Fuck Ups Game'
- 모든 팀원에게 아이디어 공유 동등 기회 → 다양한 관점 → 혁신 + 참여도 급상승
- 안전감 부재 → 두려움 기반 행동 → 창의적 억압 → 높은 퇴사율

**리더 실천 프레임워크 (Do/Don't)**:
- ✅ 작은 성과도 빠르게 인정 / 성장 지향적 피드백 / 경청과 의견 존중
- ❌ 마이크로매니징 / 인격 폄하성 질책 / 대화 독점

## 케이스 스터디: 문화가 결과를 결정한다

**슈퍼자이언트 게임즈 — 지속 가능한 성공**
- 엄격한 **노 크런치** 철학 + 스코프 의도적 제한
- '멍청한 것들(Dumb stuff)'을 의도적으로 게임에 포함 = 심리적 안전감의 직접 산물
- 그렉 카사빈: 《하데스》의 죽음·환생 루프 ← 반복적 실패를 포용하는 내러티브로 설계
- 이직률 0% + 연속 비평적·상업적 대성공

**밸브 — 수평 구조의 역설**
- 관리자·지휘체계 없음, 책상 굴리기(Desk rolling)로 자유 참여
- 결과: **그림자 계층 구조** 발생, 스택 랭킹의 '최신성 편향' 최적화 문화
- 공식 리더십 부재 → 암묵적 편견이 지배 → 다양성 저해·심리적 안전감 훼손
- 교훈: 관료제 철폐 ≠ 자동으로 좋은 문화

**CDPR 사이버펑크 2077 — 관리 실패**
- 마케팅·투자자 마감일을 실제 기술 완성도보다 우선시 → 크런치 강요
- 출시 후 직원 회의에서 분노 폭발 → 리더십-실무자 신뢰 완전 파괴
- 교훈: 재무적 마감일이 팀 역량보다 우선될 때 문화는 붕괴한다

> 💡 **핵심 인사이트:** 스튜디오 문화는 리더의 변덕에 흔들리지 않아야 한다. 원칙이 문서화되어 있을 때 리더가 바뀌어도 팀의 방향이 유지된다. 좋은 문화는 만들어지는 것이지, 자연발생하는 것이 아니다. 그리고 심리적 안전감이 없는 곳에서는 어떤 원칙도 살아남을 수 없다.
