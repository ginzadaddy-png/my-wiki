[[catalog-economics|카탈로그 이코노믹스]]가 산업 표준 패턴인 동시에 *메커니즘 분기*가 존재함을 가장 명확히 보여주는 3사 비교. 캡콤(*재발매형*), 닌텐도(*가격유지형*), Take-Two/[[rockstar-games|록스타]](*서비스형*)는 같은 "장기 매출 자산" 목표를 *완전히 다른 메커니즘*으로 실현. 한 페이지에서 trade-off를 압축.

> 💡 **핵심 인사이트:** 세 회사는 *catalog 매출 비중 70%+*라는 공통점에도 불구하고, *어떤 메커니즘으로 그 비중을 만들었는가*가 완전히 다르다. 캡콤은 *시리즈 통합 마케팅 + 리메이크 사이클*, 닌텐도는 *플랫폼 attach + 정가 유지*, Take-Two는 *단일 IP의 RCS recurrent + 재발매*. **세 모델은 호환되지 않고, 회사 DNA에 종속**된다. 한국 게임사가 catalog economics 진입을 고려할 때 *어느 모델을 모방할 것인가*가 IP·플랫폼·조직 결정으로 직결.

## 한 페이지 비교 표

|---|---|---|---|

## 모델별 메커니즘 해부

### 1. 캡콤 — *재발매형 + Cross-sell형* 결합

**핵심 메커니즘:** *시리즈 통합 마케팅*. 신작 출시가 시리즈 전체 인지도를 자극해 *카탈로그가 동반 점프*.

- RE Requiem 출시 분기에 RE4·Village·3·2·6·5가 *동시에 200~370만* 판매
- RE 시리즈 *3년 리메이크 사이클*: RE2(2019) → RE3(2020) → RE4(2023) → RE Requiem(2026 예정) — 풀프라이스 리메이크 양산이 신작 카탈로그 갱신
- *디지털 93%* + [[proprietary-engine-strategy|RE ENGINE 자체]] = *마진 극대화* — 13년 연속 영업이익 성장
- 글로벌 244개국 판매 — 아시아 5년 +247%, 중남미 +148%

**약점:** 시리즈가 식으면 *catalog 동반 슬립*. 단일 IP 의존도는 낮지만(5+ 시리즈), *세대 교체 R&D 인력*([[dev-talent-pipeline|개발 인재 파이프라인]]) 필수.

### 2. 닌텐도 — *가격유지형*

**핵심 메커니즘:** *플랫폼 attach rate + 정가 유지*. Switch 8년 단일 platform의 152M 본체에 1st-party 게임이 attach.

- Mario Kart 8 Deluxe (2017-04): FY25 누적 **68.20M**, 본체 attach rate ~45%
- *세일 최대 33%* — 정가 $59.99 8년 유지. *long-tail에서 가격이 떨어지지 않는 catalog* — 산업 내 유일
- Booster Course Pass DLC: Switch Online 가입자 무료 + 단독 $24.99 — *구독 link*
- Pokemon 시리즈 본편 **480M+** — cross-media (카드·애니·영화·GO) 효과
- Switch 2 (2025-06) cross-gen 후에도 Deluxe 판매 지속

**약점:** *플랫폼 사이클*에 종속. Switch 후 platform 전환 시 catalog reset risk. 신규 IP 진입이 *Mario·Zelda·Pokemon* 외에는 어려움 — Splatoon이 거의 유일한 신규 메가 IP.

### 3. Take-Two / [[rockstar-games|록스타]] — *서비스형 + 재발매형*

**핵심 메커니즘:** *단일 IP의 RCS recurrent + cross-gen 재발매*. GTA V 하나가 13년간 압도적 carry.

- GTA V (2013-09): FY25 Q3 누적 **210M+ units**, *13년차에도 분기당 5M units 출하*
- 3회 cross-gen 재발매: PS3/X360 → PS4/XB1 (2014) → PS5/XSX (2021)
- *GTA Online RCS*: Take-Two FY24 net bookings의 **78%**가 recurrent (GTA Online + NBA 2K MyTeam). Shark Cards 외부 추정 누적 매출 $8B+
- Epic Store 무료 배포 (2020-05) → 7M+ 다운로드 추가
- GTA 6 (2026 예정) 발표 후 *GTA V 재구매·신규 진입* ↑

**약점:** *단일 IP 의존도 risk*. GTA 6 출시 후 GTA V 카탈로그 어떻게 carry할지 미지수. NBA 2K도 RCS 의존하지만 GTA에 비하면 작음.

## 의사결정 흐름 — 어느 모델로 진입할 것인가

```
시리즈 IP 자산 보유?
   ├─ YES (5+ 시리즈) → *캡콤 모델* (재발매형 + Cross-sell)
   │      필요 조건: 리메이크 사이클 + 자체 엔진 + 글로벌 유통망
   ├─ 메가 IP 한두 개 + Platform 통제 → *닌텐도 모델* (가격유지형)
   │      필요 조건: 1st-party platform + attach rate 메커니즘
   └─ 단일 메가 IP + 라이브 서비스 결합 → *Take-Two 모델* (서비스형)
          필요 조건: Online infrastructure + RCS 마이크로트랜잭션
```

세 모델은 *상호 호환되지 않음*. 캡콤이 닌텐도식 가격유지를 시도하면 시리즈 cross-sell이 망가지고, 닌텐도가 Take-Two식 RCS를 도입하면 *Mario IP의 가족 친화 포지셔닝*이 흔들린다.

## 한국·중소 스튜디오 적용 시사점

| 모델 | 적용 난이도 | 진입 조건 |
|---|---|---|
| 캡콤 재발매형 | **매우 높음** | 시리즈 IP 5+ 누적 + 자체 엔진 + 글로벌 유통 |
| 닌텐도 가격유지형 | **불가능** | 1st-party platform 보유가 절대 조건 |
| Take-Two 서비스형 | **중간** | 단일 메가 IP + Online infrastructure |

→ 한국 게임사 중 [[nexon|넥슨]] 메이플스토리·던파, NCSoft 리니지의 *MMO recurrent 모델*은 *별개의 5번째 모델* (라이브 MMO catalog). 본 3사 모델과 다른 카테고리로 분류 필요. 인디·중견의 *현실적 진입로*는 *catalog 자체보다 시리즈화 결정* — [[supergiant-games|슈퍼자이언트]] 하데스1→하데스2처럼.

## 시간축 진화 — 어떤 모델이 가장 길게 갈까

세 모델의 *지속 가능성*은 *플랫폼 의존도 vs IP 의존도*의 함수:

- **캡콤** — 시리즈 5+개의 IP 분산 → *플랫폼 변화에 강함*. RE·MH가 Switch·Steam·PS·Xbox 멀티 플랫폼 모두 적용
- **닌텐도** — 1st-party platform 종속 → *Switch → Switch 2 전환*이 critical. 실패 시 catalog reset
- **Take-Two** — 단일 IP 종속 → *GTA 6가 GTA V를 대체*하는 자기잠식 risk

> ⚠️ **공통 risk:** *세 모델 모두 신작 출시 곡선이 long-tail 없이 우하향이면 catalog economics 진입 불가능*. 즉 *신작이 카탈로그가 될 자격*을 갖춰야 한다. 캡콤은 시리즈 통합 마케팅·리메이크 사이클로, 닌텐도는 평생 플레이 가능한 design philosophy로, Take-Two는 *Online infrastructure*로 그 자격을 만든다.
