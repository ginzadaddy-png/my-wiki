## 1. 샌드폴 인터랙티브 — "구조가 인력을 대체한다"

~40명, $1,000만 미만, 2025 TGA 9관왕 (클레어 옵스큐어: 33원정대)

- **도구 중심 구조**: 프로그래머 4명이 도구 제작 → 디자이너 전원이 콘텐츠 직접 조립
- UE5 블루프린트 95%, C++ 최소화, 엔진 무수정 → 비용 최소·업그레이드 자동
- 핵심 교훈: 팀원 수가 아니라 팀 구조가 생산량을 결정한다

---

## 2. 엠바크 스튜디오 — "F2P 환상을 버려라"

~250명, ~$7,500만, 아크 레이더스 동접 96만 (넥슨 역대 최대)

- F2P 포기 이유: "타이머·노가다는 유저 시간 기만" (디자인 디렉터 Virgil Watkins)
- $40 프리미엄으로 전환 → 2개월 유지율 91%
- 이후 -80% 하락 대응: Escalation 로드맵, "10년 게임" 선언
- CEO: 초기 판매만으로 수년간 재무 안정 — 초기 론칭이 AAA 전략의 핵심

**비전 기반 개발 병행**: 4개 기둥(높은 판돈·선택과 주도권·충실도·접근성)으로 팀 정렬

---

## 3. 애로우헤드 — "라이브 서비스를 재발명하라"

헬다이버스 2, 2,000만장+, Steam 역대 최다 동접 중 하나

- **게임 마스터 '조엘'**: TRPG 던전 마스터처럼 전쟁 전황 실시간 조작 → 뉴스가 되는 업데이트
- **비약탈적 모델**: 워본드 기간 제한 없음 + 슈퍼 크레딧 인게임 파밍 가능
- 실패 교훈: 게임 마스터 권한 남용(임의 너프, 행성 조작) → "적대적 DM 증후군", 커뮤니티 반발
- 핵심 차별화: $40 + F2P 없음 + 게임 마스터 = 새로운 라이브 서비스 공식

---

## 4. Avalanche Software (호그와트 레거시) — "AAA 전환의 4가지 함정"

32개 게임 개발사가 처음 만든 대형 오픈월드 RPG. GDC26 강연 (Kelly Murphy)

- **함정 1: 낯선 기대치** → 557개 미션 계획 → 출시 130개. 해법: 팀 강점에 야망 집중
- **함정 2: 처음 만드는 연결된 게임** → 자율적 팀이 흩어짐. 해법: "감식안(Connoisseurs)" 개발 (레퍼런스 게임 20시간 플레이 의무)
- **함정 3: 로드맵 부재** → 기능 단위로만 봄. 해법: "Popsicle Sticks" 시각화, 리드가 집중 영역 투표
- **함정 4: 벽돌만 쌓음** → 연결 경험(모르타르) 간과. 해법: 다양한 플레이스루 렌즈로 정기 점검

---

## 5. 팀 아소비 — "65명의 GOTY 공식"

소니 퍼스트파티지만 인디적 방법론. 아스트로봇 2024 GOTY

- 2주 프로토타이핑 → 아이디어 90% 초반 폐기 → 살아남은 것만 제작
- 12시간 게임에 중복 게임플레이 0, 컷신 12분, 단어 5천 개 미만
- 규모 확장 없이 품질 유지 → "완성도가 규모보다 우선"

## 수렴 원칙: 중소→AAA 공통 공식

> 💡 **핵심 인사이트:** 2020년대 중반 AAA를 달성한 중소 스튜디오들의 공통점은 "더 많은 사람"이 아니라 "더 날카로운 결정"이었다.

**1. 타겟 뾰족하게 — 모두를 위한 게임은 없다**
- 샌드폴: 턴제 RPG 팬 + JRPG 미학
- 엠바크: 익스트랙션 팬, $40 프리미엄 수용층
- 팀 아소비: 플랫포머 마니아

**2. 도구 또는 시스템이 인력을 대체**
- 샌드폴: 블루프린트 도구화 → 콘텐츠 생산 곱셈
- 엠바크: 기둥 시스템 → 팀 자율 해결
- 애로우헤드: 게임 마스터 → 업데이트 비용 없는 이벤트

**3. 손해 볼 것 같은 선택이 오히려 승리**
- 엠바크: F2P 포기 → 더 높은 유지율
- 샌드폴: $50 → 600만장 (기대 초과)
- 헬다이버스2: 비약탈 모델 → 역대급 성과

**4. $30~50 미드 프라이스가 새 스위트스팟**
- $70 AAA 공식에서 벗어남
- 게이머 22%: 핵심 게임플레이 최우선 (Bain & Company)
- 고품질 그래픽·오디오 최우선: 7%에 불과
