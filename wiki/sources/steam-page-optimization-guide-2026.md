# Steam 스토어 페이지 최적화 가이드 (2026)

**출처**: https://presskit.gg/field-guides/steam-page-optimization-guide
**날짜**: 2026

## 핵심 요약

스팀 스토어 페이지 = 주요 판매 도구. 위시리스트 대부분이 데모 없이 페이지 탐색만으로 결정됨. 최적화는 배율기(multiplier)이지 구원자가 아님.

## 캡슐 아트

| 종류 | 크기 | 위치 |
|------|------|------|
| Header | 920×430px | 페이지 상단 |
| Small | 462×174px | 검색 결과·목록 |
| Main | 1232×706px | 홈 캐러셀 |
| Vertical | 748×896px | 시즌 세일 |

규칙: 최소 크기에서 로고 가독성 확보 / 캡슐에는 아트워크+제목만 (리뷰 인용·수상 경력 금지)

## 스크린샷

- 8–12장 권장 (최소 5장 필수)
- 첫 번째 스크린샷 = 검색 결과·호버 프리뷰에 노출 → 핵심 게임플레이 표시
- 최소 4장은 전체 연령가 지정 필요

## 트레일러

- 첫 5초 이내 게임플레이 시작
- 최적 길이: 60–90초
- 무음 상태에서도 이해 가능해야 함
- **사례**: 게임플레이를 18초→1초로 앞당긴 후 위시리스트 전환율 +31% (2주)
- 스팀이 첫 트레일러에서 6초 마이크로트레일러 자동 생성

## 태그 전략

- 20개 슬롯 전부 사용
- 상위 5개가 알고리즘 가중치 최대
- "Indie", "Singleplayer" 같은 범용 태그는 상위 슬롯 금지
- 성공한 유사 게임 5–10개의 태그 분석 후 기준 설정

## 설명문

- 짧은 설명 (300자): 장르/훅 + 핵심 메카닉
- 긴 설명: GIF 리드 → 한 문단 피치 → 볼드 헤더 기능 목록 → 추가 GIF 3–4개

## 주요 수치

- 2024년 스팀 출시 게임 중 1,000개 리뷰 달성: **2.4%**
- 오픈월드 서바이벌 크래프트 성공률: 24.5% / 2D 플랫포머: 0.25%
- 스팀 유저 60%+ 비영어권 → 스토어 페이지 현지화만으로 가시성 향상

> **핵심 인사이트:** 트레일러 첫 화면을 게임플레이로 바꾸는 것만으로 전환율이 30% 이상 오를 수 있다. 페이지 최적화는 개발과 동시에 시작해야 한다.
