발표자: Anne-Marie Grønroos (Lead Mission Designer, Remedy Entertainment). GDC 2024.

Alan Wake 2의 두 주인공(Saga/Alan), 두 세계(현실/Dark Place) 구조에서 미션 설계가 어떻게 이루어졌는가.

## 기본 설계 목표

- 게임플레이와 내러티브를 긴밀하게 연결
- 피로를 방지할 충분한 다양성과 깊이
- 공포 경험
- 퍼즐·내러티브·전투를 혼합
- 핸드홀딩 없음 / 비선형 구조

## 루핑 진행(Looping Progression)

새로운 구역을 소개한 뒤 플레이어가 이전 구역으로 되돌아와 새 경로를 잠금 해제하는 구조.

| 유형 | 설명 |
|------|------|
| 비선형 | 동시에 여러 목표, 어떤 순서로도 완료 가능 |
| 선형 | 정해진 순서, 공간 재사용 |
| 반선형 | 크리티컬 패스는 선형이나 선택 콘텐츠로 공간 재방문 |

> 수사(investigation) 장르에서 선형 크리티컬 패스도 플레이어 에이전시의 환상을 만들 수 있다.

## 미션 비트 구성

내러티브 문서에서 NPC 아크와 보스 아크가 교차하며, 미션 디자이너가 요소의 배치 순서를 결정.

비트 유형: **수사 → 공포 시퀀스 → 퍼즐 → 전투**

**페이싱 전략**:
- 공포: 오래 끄는 빌드업, 분위기와 예상 집중. 루핑으로 재방문 공간의 상태 변화 가능.
- 중간 지점: 친근했던 구역이 적대적으로 변하는 상태 전환
- 대비: 기이한 낮 장면 vs 무서운 밤 장면

## 꿈의 논리(Dream Logic) 설계

**실패한 시도**: Dark Place를 순수하게 초자연적·환상적으로 만들려 했으나 정신적으로 소진됨.

**성공한 접근 — 그라운딩**:

| 좋은 형용사 | 나쁜 형용사 |
|------------|------------|
| Uncanny (섬뜩) | Fantastical (환상적) |
| Liminal (경계적) | Whimsical (기발한) |
| Ambiguous (모호한) | |
| Quirky (독특한) | |
| Ominous (불길한) | |

Remedy 게임은 스펙트럼의 "그라운드" 쪽에 위치. 꿈의 논리는 **가르침, 균형, 그라운딩**이 필요하다.

## 캠페인 구조

```
선형 시작(SAGA) →
  Wake의 Dark Place 미션 ↔ Saga의 현실 미션 (플레이어 주도 전환)
→ 선형 끝(WAKE + SAGA)
```

- Dark Place 미션이 너무 많으면 정신적으로 소진됨
- Saga의 현실 미션이 Dark Place 미션을 그라운드시킴
- Dark Place의 가장 긴 미션 ↔ 현실의 가장 긴 미션이 서로 미러링

## 핵심 교훈

1. 게임플레이와 내러티브 연결 → 디자이너·작가가 초기부터 협업, 서로의 크래프트를 위한 빈 공간 남기기
2. 일관성과 깊이 → 같은 목표를 지지하는 여러 요소
3. 다양성 → 서로 다른 요소, 대비, 페이싱에서
4. 이미 배운 것을 활용 → 플레이어의 시간과 에너지 절약
5. 원래 설계 목표를 놓을 줄 알아야 함
6. 가능성 공간은 생각보다 크다
