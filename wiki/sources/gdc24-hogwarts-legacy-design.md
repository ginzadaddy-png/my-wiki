GDC 2024, 아발란체 소프트웨어(Avalanche Software) 켈리 머피 리드 게임 디자이너 강연. 출처: Inven

## 배경: 이중 타겟의 딜레마

- 위저딩 월드 팬 대부분은 **비게이머** — 호그와트 레거시 콘셉트에 큰 관심을 보였으나 구매력·플레이 기반 미증명
- AAA 규모의 게임이라면 기존 게이머도 잡아야 함
- 결론: **팬(Fan) + 플레이어(Player)** 두 집단 모두를 만족시키는 게임 필요

## 두 집단의 특성 차이

| 항목 | 플레이어 | 팬 |
|------|---------|-----|
| 게임 경험 | 많음, RPG 익숙 | 매우 적음 |
| 선호 콘텐츠 | 긴 분량, 시스템 최적화, 전략 | 스토리 기반, 감정적 경험, IP |
| 시스템 의존성 | 연계 구조 즐김 | 복잡한 연계에 거부감 |
| 운(무작위) 요소 | 가장 중요하게 여김, 적극 투자 | 극단적으로 싫어함, 실패에 집중 |
| 재능 선택 | 수치 차이 미세하게 고민 | 재능을 거의 사용하지 않음 |
| 네비게이션 | 전체 지도·미니맵 즉시 활용 | 방향 파악 못 하고 길을 잃음 |

## 5가지 해결 접근법

### 1. 시스템 의존성 분해

일반 AAA RPG의 시스템 연결 고리는 약 **345개** (전투→강화→생활콘텐츠 순환).  
아발란체는 이 연계를 **의도적으로 분해** → 각 콘텐츠를 최소한의 의존성만 남겨 원하는 것만 즐길 수 있게.  
플레이어형 풀과 팬형 풀을 분리해 운영.

### 2. 무작위성(운 요소) 풀 분리

운 요소를 완전히 제거할 수 없었음 (플레이어층이 선호하기 때문).  
→ **플레이어 풀**: 운 요소 적극 적용 / **팬 풀**: 운 요소 완전 제거

### 3. 재능 시스템 단순화: 180개 → 40개

초기 재능 180개 → 플레이어는 미세 수치(5%+5%+5%)까지 고민하지만, 팬은 재능 자체를 거의 안 씀.  
해결: 같은 효과 재능을 하나로 통합 (인센디오 5%×3 → 인센디오 15%×1).  
추가 효과를 주는 재능도 하나만 올려도 전체 효과 적용.  
→ 약 **2년에 걸쳐 40개**로 압축.

### 4. 네비게이션: 포인트 → 경로

- 게임 디자이너(플레이어형)는 세계를 "포인트의 집합"으로 봄
- 팬은 세계를 "일련의 지도"로 보고 경로를 연결하려 함 → 웨이포인트 방식에서 길을 잃음
- 해결: **기존 웨이포인트 제거** → 경로 자체를 보여주는 "매혹적인 내비게이션" 도입

### 5. 난이도: 비게이머도 클리어 가능한 수준까지

- 출시 전 **12개월** 동안 난이도 조정이 가장 큰 고통
- 팬 그룹 반응: "어렵다, 어렵다, 또 어렵다"
- 동시에 팬들은 게임을 친구·가족·파트너와 공유하고 싶어함
- 해결: 게임에 관심 없는 이들도 클리어 가능한 수준의 난이도 옵션까지 제공

> **핵심 인사이트:** IP 게임의 타겟은 단일하지 않다. 팬과 게이머는 같은 게임에서 전혀 다른 것을 원한다. 핵심은 **시스템을 강제 연계시키지 않고 분리해 각자가 원하는 방식으로 즐길 수 있게 설계**하는 것.
