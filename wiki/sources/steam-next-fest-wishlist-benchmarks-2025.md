# Steam Next Fest 위시리스트 벤치마크

**출처**: https://howtomarketagame.com/2025/03/26/benchmarks-how-many-wishlists-can-i-get-from-steam-next-fest/
**저자**: How To Market A Game (Chris Zukowski)
**날짜**: 2025-03

## 핵심 요약

182개 게임 설문 기반 정량 분석. 사전 위시리스트 수가 행사 성과의 가장 강력한 예측변수(r=0.825).

## 성과 티어

| 티어 | 기준 |
|------|------|
| Diamond | 10,000+ |
| Gold | 7,000–9,999 |
| Silver | 1,000–6,999 |
| Bronze | 0–999 |

## 사전 위시리스트 vs 행사 성과

| 사전 위시리스트 | 행사 중 획득 위시리스트 (중간값) |
|---------------|-------------------------------|
| 0–999 | 322 |
| 1,000–9,999 | 1,006 |
| 10,000–99,999 | 5,215 |
| 100,000+ | 12,882 |

## 데모 전환율

- 공식: 데모 플레이 + 위시리스트 등록 ÷ 전체 플레이어
- 적정 수준: ~20%
- 상관관계: r=0.457 (보통 수준)
- 낮은 전환율 = 게임 품질 문제보다 프레젠테이션 미스매치 가능성

## 알고리즘 패턴

3일 차부터 노출이 상위 게임에 급격히 집중. "밸브가 좋아하는 게임이 되거나, 아무것도 아니거나."

> **핵심 인사이트:** SNF 성공의 핵심은 행사 당일이 아니다. 최소 2,000개 이상의 사전 위시리스트를 갖추고 입장해야 알고리즘 경쟁에서 유의미한 위치를 점할 수 있다.
