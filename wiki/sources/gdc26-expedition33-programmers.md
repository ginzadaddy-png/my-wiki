GDC 2026에서 [[sandfall-interactive|샌드폴 인터랙티브]] Tom Guillermin(공동창업자)과 Florian Torres(시니어 프로그래머)가 공개한 4인 프로그래머 팀의 개발 방식.

**핵심 내용:**
- **프로그래머 4명**으로 대형 RPG 완성 (총 ~40명 스튜디오)
- **게임 로직 95% 블루프린트**: 디자이너도 편집 가능. 에디터 안정성·자동완성으로 온보딩 용이
- **엔진 무수정 원칙**: 버전 업데이트를 출시 직전까지 유지 → 버그 수정·최적화 무상 획득
- **빌드 서버 구축**: 패키징 시간 2시간 30분 → 50분으로 단축
- **프로그래머 = 도구 제작자**: 콘텐츠는 테크니컬 디자이너가 100% 담당. 통합 API + 에러 즉시 팝업 강제
- **전투 스킬 90%가 레벨 시퀀서**: 디자이너가 애니메이션·카메라·이펙트·대미지를 직접 조립

> **핵심 인사이트:** 소규모의 해답은 '덜 만드는 것'이 아니라 '더 잘 나누는 것'. 프로그래머가 콘텐츠를 만들지 않는 구조가 핵심.
