GDC 2024, 마케팅 컨설턴트 크리스 주코프스키(Chris Zukowski) 강연. 밸브 출신이 아닌 외부자가 수집한 스팀 알고리즘 데이터. 출처: Inven

## 빅 시크릿 3가지 (기반 명제)

1. **스팀에는 혼자서 할 수 있는 것보다 더 많은 노출 채널이 있다** — SNS·틱톡보다 스팀 내 노출이 압도적
2. **스팀 상점은 사실상 2개다 — '레알 스팀'은 따로 있다** — 전체 게임의 상위 6%만 입점. 여기서 매출 대다수 발생
3. **스팀은 큐레이트 한다 — 단, 요청해야만** — 요청 안 하면 절대 먼저 해주지 않는다

> 스팀웍스 서포트 페이지에 숨겨진 정보가 많다. 설명서를 읽지 않으면 기회를 놓친다.

## 대상 1: 초기 인디 전략

### ① 스팀 페이지를 번역하라

번역 우선 순위: **중국어(간체) → 일본어·한국어 → FIGS(불·이·독·스)**  
번역 최소 범위: 스팀 페이지 + 데모 (게임 전체가 어려우면 이것만이라도)  
미번역 시 스팀이 해당 언어 사용자에게 경고 문구 표시 → 구매 전환율 하락

### ② 스팀 첫 페이지에 이름 올리기

'신규 출시 인기 제품' 또는 '인기 출시 예정 제품' 진입이 목표. 일당 1,000개 이상 찜하기 보장.

**세부 팁:**

| 팁 | 내용 |
|-----|------|
| **출시 분(分) 설정** | X시 1분 또는 59분 — 스팀 갱신 시간 기준으로 경쟁자보다 앞 번호 획득 가능 |
| **월요일 출시 강추** | 주말(금·토·일) 3일 동안 '인기 출시 예정 제품' 최상단 유지. 화요일은 시스템 오류 多 |
| **퍼블리시 버튼 필수** | 출시 시 '퍼블리시 나우' 미클릭 시 누적 성과가 재산정됨 → 8,000달러/일 기준 초기화 |
| **AAA와 같은 날 출시** | 경쟁 피하려 연기된 게임들로 '신규 출시' 슬롯에 공간 생김. 사례: 칠쿠아리움이 스타필드 다음날 출시 → 최고 인기 3위 달성 |

## 대상 2: 레알 스팀 (상위 6%, 약 26만 달러 이상)

### ① 밸브 프로모션 제안이 오면 무조건 YES

할인이 손해처럼 보여도 노출 이익이 훨씬 크다.

| 딜 종류 | 기간 | 기대 매출 |
|---------|------|----------|
| 데일리 딜 | 24시간 | 최소 25만 달러 |
| 미드위크 딜 | 72시간 | — |
| 위크엔드 딜 | 96시간 | 최소 100만 달러 |

→ 첫 페이지 할인 구간 배치를 **강하게 요청**할 것 (말 안 하면 안 해줌)

### ② 다양한 딜 트리거 활용

신규 콘텐츠·언어 추가·후속작 출시·최저가 책정 등도 딜 생성 가능.

### ③ 출시 할인을 시즌 세일과 연결

출시 할인이 '종료'된 후 30일간 추가 할인 불가.  
→ **출시 할인 기간을 여름 세일 등 시즌 세일과 겹치도록** 설정하면 제한 없이 연속 할인 가능.

### ④ 페스티벌 개인 페이지 생성

스팀웍스 → '세일 & 이벤트' → '오거나이즈 유어 원 세일즈'  
대형 퍼블리셔처럼 자체 페스티벌 페이지 구성 가능.

## 대상 3: 메가 파퓰러 (상위 2%, 60만 달러+, 10만 찜 이상)

- **프랜차이즈 할인 요청**: 후속작 공개만으로도 프랜차이즈 페이지 생성 가능. 구작 할인 + 신작 노출 동시에
- **스팀 팝업 요청**: 스팀 켤 때 자동 팝업 윈도우에 게임 노출. PSD/JPG 파일 제출 시 약 4시간 후 런칭
- **번들 구성 요청**: 다른 개발사 게임과 묶기 가능. 사례: 페글린 + 라운드가드 번들 — 페글린 122,450달러, 라운드가드 70,021달러 추가 매출

> **핵심 인사이트:** "스팀의 알고리즘은 항상 변한다. 하지만 바뀌기 전까지는 충분히 써먹을 수 있다." 핵심은 **스팀에 요청하는 것** — 알고, 요청해야만 기준을 충족하면 실행해준다.
