[[nintendo|닌텐도]]의 대표 게임 디자이너·프로듀서. 마리오·젤다·동키콩·피크민·스타폭스 등 닌텐도 핵심 IP의 창시자. 인더스트리얼 디자인(ID) 전공으로 입사해 게임 디자인을 독학, 패미컴 시대부터 현재까지 닌텐도 개발의 중심.

| 항목 | 내용 |
|------|------|
| 직책 | Representative Director, Fellow (닌텐도) |
| 출신 | 인더스트리얼 디자인 (가나자와 미술공예대학) |
| 대표작 | 슈퍼 마리오, 젤다의 전설, 동키콩, 피크민, 스타폭스 |
| 협업 | 故 이와타 사토루(전 사장), 이토이 시게사토(『MOTHER』) |

## 개발 철학

**최소 인원 시작**: 대형 프로젝트도 5명으로 시작하고 30명을 넘기지 않으며, 초기 팀에 디자이너를 일부러 넣지 않아 의존·지연을 막는다. → [[small-team-development|소규모 팀 개발]]

**"아무것도 할 수 없어서 프로듀서가 됐다"**: 팀이 클수록 자신이 빈틈을 메우는 역할이 되며, 자신이 할 일이 없는 팀이 최고의 프로듀싱. → [[producer-role|프로듀서 역할]]

**손맛 우선**: 조작했을 때의 감각(마리오 착지 80% 속도 유지 등)을 프레임 단위로 직접 챙긴다. 게임을 "예술"보다 "과학과 기술"로 본다. → [[game-feel|게임 필]]

**제약 = 깊이**: 하드웨어 제한을 완벽히 이해할 때 비로소 "조각 퍼즐" 같은 밀도가 나온다. → [[constraint-driven-creativity|제약 기반 창의성]]

**홈런 지향**: 칭찬을 아끼고 진짜 보상은 대히트(3천만 장)라 믿으며, 손익분기 사고를 거부하고 "힘껏 배트를 휘둘러라". → [[hit-driven-strategy|홈런 전략]]·[[audience-expansion|오디언스 확장]]
