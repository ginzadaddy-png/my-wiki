일본 교토 소재 게임 회사. 마리오·젤다·포켓몬 등 세계적 IP 보유. 독자 하드웨어(닌텐도 스위치)와 소프트웨어를 모두 개발.

| 항목 | 내용 |
|------|------|
| 설립 | 1889년 (게임 사업은 1970년대~) |
| 위치 | 일본 교토 |
| 대표 IP | 젤다, 마리오, 포켓몬, 메트로이드 |
| 개발 철학 | 시스템 설계 우선, 창발적 게임플레이 |

## 개발 철학

**창발적 시스템 설계**: "재밌는 것을 만들기보다 재미있는 일이 일어나도록 만드는 시스템을 만드는 것." 티어스 오브 더 킹덤에서 물리 법칙 일관 적용으로 개발진이 예상 못 한 플레이가 자동 생성됨 → [[emergent-systems-design|창발적 시스템 설계]] 참고

**전용 구현 거부**: 차량·사운드·퍼즐 모두 개별 스크립트 없이 물리 규칙에서 창발. 오브젝트마다 따로 코드를 짜는 대신 모든 오브젝트에 동일한 규칙을 적용.
