1인 개발자 로컬성크(LocalThunk)가 만든 포커 기반 로그라이크 카드 게임. 퍼블리셔: 플레이스택(Playstack).

| 항목 | 내용 |
|------|------|
| 개발사 | 로컬성크 (1인) |
| 퍼블리셔 | 플레이스택 |
| 장르 | 로그라이크 / 카드 |
| 수상 | 더 게임 어워드 3관왕, GDC 2025 4관왕 |

## 특징

- 비주얼이 미려하지 않음 — 스크린샷·트레일러만으로는 매력 전달 어려움
- 게임성 자체(한 번 잡으면 멈추지 못하는 루프)가 핵심 셀링 포인트
- 도박 논란으로 일부 국가 스토어 삭제 → 오히려 화제 전환

## 마케팅 역사

데모 3회 + 인플루언서 전략 + 커뮤니티(디스코드 1만 명)로 주목의 수레바퀴 완성.
