벨기에 겐트 본사의 RPG 전문 스튜디오. 유럽·북미·아시아 다국적 운영.

- **대표**: 스벤 빈케(Swen Vinke)
- **역사**: 디비니티: 오리지널 신 1편(300만 장) → 2편(800만 장) → [[baldurs-gate-3|발더스 게이트3]]
- **특징**: [[early-access-strategy|얼리 액세스 전략]] 일관 활용, 글로벌 22시간 개발 체제
- **BG3 이후**: D&D 라이선스 반납, 오리지널 IP로 전환 선언
- **마스코트**: 반려견 구스타프 (BG3 프로젝트명, 게임 내 다수 명칭에 반영. 강연 중 별세 추모)

## BG3 EA 전략 — 검증 도구로서의 EA

[[vincke-bg3-ea]] (GDC 2024 + 빈케 공개 발언 종합):

- **3년 EA (2020-10 → 2023-08), 액트 1만 공개**. 액트 2~3는 스토리 서프라이즈 보존 위해 1.0까지 비공개.
- EA 동안 **액트 1을 텔레메트리 기반으로 여러 차례 재편집** — 검증 루프로 활용.
- EA 250만 카피 판매. 빈케는 "BG3가 EA에서 정점을 찍었을지" 출시 전 우려한 적 있음.
- **EA 모델 옹호**: "EA를 성공적으로 졸업한 게임은 모델을 남용하지 않는다. BG3가 3년 동안 EA였다는 걸 알지?" (2025-03 발언)
- BG3 = "시작·중간·끝이 있는 완결작" → 시퀄·확장팩 없음 명시. WOTC가 IP 보유.
